# vue.js Setup

1. 기본 설치 및 설정
; VSCode 설치: Visual Studio Code 공식 웹사이트에서 OS에 맞는 버전을 다운로드하여 설치합니다.
; Node.js 설치: Vue 개발에 필수적인 Node.js와 npm을 설치합니다.
; Volar 확장 설치: VSCode를 열고 Extensions 뷰에서 'Volar'를 검색하여 설치합니다. 이는 Vue.js 개발을 위한 필수 확장입니다.
; Vue CLI 설치: 터미널에서 npm install -g @vue/cli 명령으로 Vue CLI를 전역 설치합니다. 

2. VSCode 확장 프로그램 추천
; Volar: Vue.js 개발을 위한 공식 확장 (SFC, TypeScript 지원).
; Prettier - Code formatter: 코드 스타일을 일관되게 유지하도록 도와줍니다. VSCode 설정에서 단축키(예: Ctrl+Alt+F)를 지정할 수 있습니다.
; ESLint: 코드 품질을 검사하고 잠재적 오류를 찾아줍니다.
; Korean Language Pack: VSCode를 한국어로 사용하려면 설치합니다. 

3. Vue 프로젝트 생성 및 실행
; 프로젝트 생성: 터미널에서 vue create my-vue-app (my-vue-app은 프로젝트 이름) 명령을 실행합니다.
; 옵션 선택: Babel, ESLint 등을 포함한 기본 프리셋을 선택하거나 수동으로 기능을 추가할 수 있습니다.
; 프로젝트 실행: 생성된 폴더로 이동하여 npm run serve 명령으로 개발 서버를 실행합니다. 

4. 디버깅 및 추가 설정
; 디버깅: Chrome 브라우저 설치 후, VSCode의 디버그 기능을 활용하여 Vue 앱을 디버깅할 수 있습니다.
; 코드 포맷팅 설정: settings.json 파일에 Prettier 설정을 추가하여 저장 시 자동 포맷팅을 활성화할 수 있습니다.
